# UI Structure

## Application Layout

Watermelon uses a workspace style interface divided into five areas:

Top Bar\
Left Sidebar\
Main Workspace\
Export Settings Panel\
Download Queue

## Top Bar

Contains: - Application logo - YouTube URL input - Analyze button -
Paste from clipboard

## Left Sidebar

Navigation items: Home Downloads Tools Settings

## Main Workspace

Contains: - Video preview player - Playback controls - Timeline editor -
Clip selection markers

## Export Panel

Controls clip export options including format, quality and filename.

## Download Queue

Displays active and completed downloads with progress indicators.
