# Application Behavior

## Purpose

This document describes how Watermelon behaves when users interact with
the application.

## Application Launch

When the application opens, the user sees the main workspace with a
YouTube URL input field and empty timeline.

## Loading a Video

1.  User pastes YouTube link.
2.  User clicks Analyze.
3.  Video preview and timeline appear.

## Video Playback

Users can play, pause, and seek through the timeline to locate moments
in the video.

## Clip Selection

Users select clip start and end points using IN and OUT markers on the
timeline.

The highlighted region represents the clip that will be downloaded.

## Export Settings

Users choose export format (MP4, MP3, GIF) and quality before
downloading.

## Download Behavior

When Download Clip is clicked: - The clip enters the download queue. -
Progress is displayed until completion.

## Download Completion

Completed downloads remain visible in the queue.
